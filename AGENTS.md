Этот проект – я придумываю тему для подачи на конкурс "Студенческий стартап", а также я хочу взять эту же тему для своей магистерской диссертации. У меня появилась такая идея: Создание системы акустического обнаружения и идентификации звукоизлучающих объектов - Наземный комплекс из массива микрофонов (4–8 шт.), определяющий по звуку моторов направление и приблизительную дальность до приближающегося дрона / другого звукоизлучающего объекта. Алгоритмы: beamforming для оценки направления прихода звука, спектральный анализ для выделения частот моторов, бинарный классификатор «дрон / не дрон» для фильтрации (птицы, машины, ветер). Результат — азимут и оценка расстояния — выводится на карту в реальном времени.
Контекст: Я на 1 курсе магистратуры КАИ по направлению "Прикладная информатика", работаю Java разработчиком, интересуюсь обучением нейросетей.
Задача: Написать магистерскую диссертацию с научной новизной
Цель: Запустить свой стартап по разработке устройств для обнаружения дронов
Ожидаемый результат: Диссертация по стандартам оформления со всеми документами в папке: dissertation
