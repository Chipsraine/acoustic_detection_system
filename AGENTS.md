Этот проект – я придумываю тему для подачи на конкурс "Студенческий стартап", а также я хочу взять эту же тему для своей магистерской диссертации.

Утверждённое название: "Создание системы акустического обнаружения и идентификации звукоизлучающих объектов"

Описание: Наземный комплекс из массива микрофонов (4–8 шт.), определяющий по звуку направление и приблизительную дальность до любого звукоизлучающего объекта — дрона, самолёта, автомобиля, животного и т.д. Тип обнаруживаемых объектов определяется задачей конкретного заказчика. Алгоритмы: beamforming для оценки направления прихода звука (DOA), спектральный анализ для выделения акустических сигнатур, многоклассовый классификатор для идентификации типа объекта. Результат — азимут, оценка расстояния и тип объекта — выводится на карту в реальном времени.

Область применения: исключительно гражданская. Примеры:
- Мониторинг воздушного движения в зонах аэропортов (несанкционированные БПЛА)
- Охрана государственной границы (гражданская таможенная/пограничная служба)
- Экологический мониторинг (обнаружение животных по крикам, браконьерство)
- Охрана периметра промышленных объектов, заповедников, складов
- Мониторинг транспортных потоков
- Системы раннего оповещения для аэродромов малой авиации

Контекст: Я на 1 курсе магистратуры КАИ по направлению "Прикладная информатика", работаю Java разработчиком, интересуюсь обучением нейросетей.
Задача: Написать магистерскую диссертацию с научной новизной
Цель: Запустить свой стартап по разработке устройств акустического обнаружения и идентификации звукоизлучающих объектов
